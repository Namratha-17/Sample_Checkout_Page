#Prerequisites:
Step 1: Node.js - https://nodejs.org/en/download/
Step 2: Open node.js command prompt
Step 3: Install angular -  npm uninstall @angular/cli
Step 4: Go to the folder path "/Sample-Checkout-Project".
step 5: Run npm install
Step 6: Run ng serve command for the project to execute.

The project will be checked on the link http://localhost:4200/
